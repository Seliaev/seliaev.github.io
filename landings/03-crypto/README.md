# 🚀 NEXUS — Крипто-платформа

> Лендинг для криптовалютной платформы с живым тикером цен, анимированными графиками и таблицей рынка.

## 🛠 Технологии
- HTML5 / CSS3 / Vanilla JS
- Canvas API — мини-чарт, основной чарт BTC, спарклайны в таблице
- CSS glassmorphism + CSS-анимации (звёздное небо, float, blink)
- IntersectionObserver — reveal и счётчики

## ✨ Секции
| Секция | Описание |
|--------|----------|
| Hero | Заголовок + floating BTC-карточка + 4 счётчика |
| Ticker | Живая лента цен 5 монет (скроллинг) |
| Возможности | 6 glassmorphism-карточек |
| Рынок | Таблица TOP-5 + спарклайны + основной чарт Canvas |
| Как это работает | 3 шага |
| Roadmap | 4 квартала с бейджами статуса |
| CTA | Email-форма + модальное окно |
