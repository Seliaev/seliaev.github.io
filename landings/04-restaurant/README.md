# 🍽️ GUSTO — Ресторан авторской кухни

> Лендинг для fine-dining ресторана с интерактивным меню, системой бронирования и галереей.

## 🛠 Технологии
- HTML5 / CSS3 / Vanilla JS
- Шрифты: Playfair Display + Jost (Google Fonts)
- Без зависимостей — работает офлайн

## 🚀 Запуск
```bash
cd portfolio/landings/04-restaurant
npx -y http-server -p 8083 -c-1
# Открыть: http://localhost:8083
```

## ✅ Секции
| Секция | Фичи |
|--------|------|
| Hero | Animated plate, awards bar |
| Stats | Animated counters |
| Menu | 4 вкладки (24 блюда/вина), анимация |
| About | Chef card, features |
| Gallery | CSS-gradient photos grid |
| Reviews | 3 карточки |
| Booking | Datepicker, guests picker, phone mask, validation, modal |
| Footer | 4 колонки |
