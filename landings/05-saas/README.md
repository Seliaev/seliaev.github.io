# ⚡ PULSE — SaaS Аналитическая платформа

> Лендинг B2B SaaS-продукта с интерактивным дашбордом, pricing toggle и живыми Canvas-графиками.

## 🛠 Технологии
- Vanilla HTML / CSS / JS
- Canvas API — 5 живых графиков (hero chart, donut, feature chart, dashboard line+donut)
- Particle system — фоновые частицы
- IntersectionObserver — scroll reveal + счётчики

## 🚀 Запуск
```bash
npx -y http-server . -p 8084 -c-1
# → http://localhost:8084
```

## 📦 Секции
- Hero с живым animated dashboard (revenue chart, donut, floating KPI cards)
- Stats bar — 4 анимированных счётчика
- Features — 5 возможностей с glassmorphism-картой и Canvas-графиком
- Dashboard preview — переключаемые вкладки (Обзор/Маркетинг/Продажи/Продукт)
- Integrations — 18 логотипов, marquee scroll
- Pricing — 3 плана + toggle помесячно/ежегодно (-20%)
- Testimonials — 3 отзыва с аватарами
- FAQ — аккордеон
- CTA — email-форма с toast-уведомлением
