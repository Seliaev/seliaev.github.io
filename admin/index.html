📨 Уведомления о заявках (Cloudflare Worker)

Заявки с формы приходят в Telegram через защищённый прокси-сервер.
Токен бота хранится на сервере Cloudflare — посетители сайта его не видят.
Вставьте URL вашего Worker (например: https://portfolio-notify.xxx.workers.dev).

ℹ️ URL Worker публикуется в data.js — это безопасно. Сам токен Telegram хранится только в Cloudflare Secrets.

🚀 Публикация на GitHub Pages

Нажмите кнопку ниже — изменения сразу появятся на seliaev.github.io.
GitHub Token хранится только в вашем браузере (localStorage), не публикуется.

ℹ️ Нужен токен с правами repo. Создайте на github.com/settings/tokens
